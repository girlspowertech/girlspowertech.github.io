你好，欢迎来到 Girls Power Tech 的主页。脱离象牙塔后职场性别歧视成了一个不可忽视的问题。新一轮技术革命带来了新的财富，如果能把这阵春风吹到偏远山区的孩子特别是女孩，哪怕是帮助一个，能让这个世界变好一点点那就足够了。Girls Power Tech 就是在做这样的事，作为一名幸运搭上了这趟车的人，这是我目前要寻的意义。

我将在这个 repo 对以下相关内容进行整理：

1. 计算机入门
     - 整理一些认为比较有效的学习、提问方法文章
     - 只涉及基础的技术指南，和一些方向性的指导。我也不愿误人子弟，更深的内容肯定需要去自学
     - 如果有可能的话，希望 issue 可以成为基础的提问训练，这也是我们的一个小小讨论空间，PR 成为首次的 git 提交训练
2. 女权主义文章
    - 整理被政府删除的互联网记忆
    - 鼓励每一个在这种环境下失意、痛苦的女性前行

十分欢迎你来提 PR 进行纠错、改善 : )
